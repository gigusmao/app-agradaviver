# Agradaviver

Este aplicativo móvel foi desenvolvido como produto prático do projeto de Iniciação Científica intitulado **"Aplicação de métodos e técnicas computacionais para desenvolvimento de aplicativo para controle de doenças crônicas: obesidade"**. O projeto foi conduzido na Faculdade de Medicina da Universidade de São Paulo (FMUSP) com fomento concedido pela Fundação de Amparo à Pesquisa do Estado de São Paulo (FAPESP).

---

## 📄 Resumo do Projeto
O avanço global da obesidade demanda soluções digitais inovadoras que transcendam abordagens puramente biomédicas. O **Agradaviver** propõe preencher essa lacuna ao focar no registro de peso e, prioritariamente, no acompanhamento de metas personalizadas de saúde. 

O grande diferencial científico e clínico da aplicação está na jornada guiada de criação de metas. Essa jornada integra ferramentas da **Entrevista Motivacional (EM)** aos princípios de **Literacia para a Saúde** e **Autocuidado Apoiado**, guiando o usuário de forma interativa através dos processos de engajamento, foco, evocação e planejamento para estimular a autonomia e a mudança sustentável de comportamento.

---

## Aspectos Técnicos e Tecnologias
O desenvolvimento seguiu a metodologia ágil **Scrum** para permitir revisões e melhorias incrementais. O ecossistema técnico do app é composto por:

* **Front-end:** React Native (Framework baseado em JavaScript para desenvolvimento multiplataforma nativo para Android e iOS).
* **Back-end & Banco de Dados:** Firebase (Plataforma do Google utilizada para autenticação de usuários, armazenamento em nuvem e sincronização de dados em tempo real).
* **Ambiente de Desenvolvimento:** Visual Studio Code (VS Code).
* **Prototipagem:** Figma (Design de interface UI/UX e mapeamento do fluxo de telas).
